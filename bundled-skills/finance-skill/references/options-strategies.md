# 期权策略（options-strategies）

> **何时参考**：用户问"期权策略 / 50ETF 期权 / 300ETF 期权 / 隐含波动率 / Delta / Gamma / 牛市价差 / 跨式 / 铁秃鹫 / 期权对冲"。

## 核心目标

**构建多腿期权组合 + Greeks 风险管理 + 波动率观点**。

## 主要内容

### 1. 期权定价基础（Black-Scholes）
- 输入：标的价 S / 执行价 K / 期限 T / 无风险利率 r（0.03-0.05）/ 隐含波动率 σ（20%-40% 常见）
- 不要用 BS 给 OTM/ITM 深度期权定价——实际隐含波动率在微笑区可能与 ATM 相差 5%-10%

### 2. Greeks 管理体系
- **Delta**：方向敞口，0 到 ±1
- **Gamma**：对冲稳定性，ATM 最高
- **Theta**：时间衰减，短期高
- **Vega**：波动率敏感性，ATM 最大
- **Rho**：利率敏感性，长期相关
- 在投资组合层面按风险贡献加总

### 3. 交易策略地图

| 观点 | 策略 | 现金流 |
|---|---|---|
| 看涨牛市 | 看涨买权 / 牛市看涨价差 | net debit |
| 看跌熊市 | 看跌买权 / 熊市看跌价差 | net debit |
| 中性区间 | 铁秃鹫 / 铁蝴蝶 | net credit，风险有限 |
| 不确定大幅波动 | long straddle（高成本）/ long strangle（较便宜） | net debit |

### 4. 波动率交易维度
- 隐含 vs 实现波动率（IV-RV 差值是套利信号）
- 波动率曲线形态（微笑 / 偏斜 / 期限结构）
- Greeks 再平衡频率：ATM 每日对冲 → OTM 周度对冲 → 极端情况按阈值

### 5. 中国市场特色
- 50ETF / 300ETF 期权流动性集中在 ATM ±3 执行价
- 近月合约成交最活跃，远月 / 深度 OTM 流动性差、滑点大
- 期权乘数（50ETF 通常 1.0，需确认）
- 欧式行权（仅到期可行权，无美式早行权价值）

### 6. 多腿组合可视化
- 回报曲线（横轴标的价、纵轴 P&L）
- Greeks 曲线（Delta / Gamma / Theta / Vega 随标的价变化）
- 同时展示多个波动率情景（σ 变化 50% / 75% / 100% / 125% / 150%）

### 7. 高级策略陷阱
- 波动率微笑导致 OTM 定价偏差——需 SABR 或局部波动率模型修正
- Gamma 风险在接近到期时快速增长——short gamma 组合在大波动下暴露
- 近月 Theta 衰减非线性——最后 30 天衰减速度远超前期

## 建议骨架（不强制）

- 策略定义（多腿结构 / 各腿方向 / 执行价 / 期限 / 数量 / 成本）
- 回报图与 Greeks 曲线（含多个波动率情景）
- 盈亏平衡点（Break-Even Points）+ 最大利润 / 损失
- 风险管理规则（止损水位 / 再平衡触发 / Greeks 上下限）

## 避坑

- 液体期权与深度 OTM 期权买卖价差相差 5-10 倍——小资金容易滑点超过预期收益
- 高 Gamma 头寸在极端行情下对冲成本爆炸——需预留充足缓冲或提前减仓
- 不要忽视近月期权的 Gamma / Theta 非线性——最后 7 天衰减最快
- 卖出未对冲期权（裸卖）风险无限——只在对冲组合内使用
- 期权流动性差时强行平仓会被市场吃 5%-10%
