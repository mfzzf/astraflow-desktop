# 交易计划与买卖点（trade-plan）

> **何时参考**：用户已选好股票，问"什么价位买 / 什么时候加仓 / 什么时候止盈止损 / 帮我做交易计划 / 盘前执行单"。

## 核心目标

**给出可实际执行的计划**——具体条件 + 具体动作，而不是"逢低布局、注意风险"这种模糊话。

## 分析步骤

1. **判断当前交易位置**：
   | 位置 | 特征 | 信号 |
   |---|---|---|
   | 低位启动 | 长期下跌后横盘 / 放量突破 | 30 日低点 + 放量 |
   | 趋势中继 | 上升趋势中回调 | 5 日线附近 + 缩量 |
   | 高位加速 | 连续大涨后加速 | 远离 5 日线 + 放量 |
   | 高位震荡 | 高位横盘 | 5 日线走平 + 高换手 |
   | 下跌反抽 | 下跌后反弹 | 5 日线下方 + 无量 |
   不同位置策略完全不同。
2. **入场依据**：必须基于（1）逻辑催化（政策 / 业绩 / 订单）（2）趋势结构（突破 / 回踩 / 反转）（3）量价配合（放量上涨 / 缩量回调）（4）市场环境（大盘 / 板块支持）。**理由不足时明确"宁可等确认"**。
3. **分仓路径**（不要一把梭）：
   - 试错仓 10-20%：首次信号
   - 确认仓 +20-30%：走势符合预期
   - 加仓仓 +20-30%：强势确认 / 突破
   - 重仓 >70%：极强势 + 环境支持
4. **止盈止损**：
   - 止损：技术止损（跌破前低 / 5 日线 / 关键支撑）或幅度止损 -5% 至 -8%（按波动率调整）或时间止损（X 天未涨）
   - 止盈：目标止盈（前高 / 阻力位）或情绪止盈（换手 >15% + 放量滞涨）或时间止盈（计划周期到）
5. **失效条件**（高质量计划必备）：催化证伪 / 板块转弱 / 龙头负反馈 / 放量破位 / 次日反馈差。**避免死扛的关键。**
6. **执行提醒**：左侧埋伏 vs 右侧确认、快进快出 vs 波段持有、是否适合重仓、开盘买 / 回调买 / 突破买。
7. **市场环境确认**：大盘支持 / 中性 / 不支持、板块主线 / 跟风 / 退潮、资金进攻 / 防守 / 观望。**环境不支持时降仓或放弃。**

## 建议骨架（不强制）

- 当前交易位置判断
- 入场逻辑与催化
- 分仓路径设计
- 止盈止损规则
- 失效条件与触发器
- 执行前检查与环境适配

## 避坑

- 不写"逢低布局、注意风险"——必须具体条件 + 具体动作
- 一把梭不是计划，分仓是标配
- 环境不支持时敢直说"不执行"
- 止损位写完就是写完，不要事后宽松
