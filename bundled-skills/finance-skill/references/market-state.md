# 市场状态与广度（market-state）

> **何时参考**：用户问"现在市场怎么样 / 该不该进 / 行情还能持续吗 / 大盘是真涨还是少数股拉的 / 风险大不大"。
>
> **输出形式**：本场景结论以 HTML 文件呈现（用 `Write` 工具落盘到当前工作目录，文件名形如 `<topic>-<YYYYMMDD>.html`），对话正文里附 200-300 字摘要 + 文件路径让用户自行打开。

## 核心目标

**判断当前市场处于哪个阶段**（进攻 / 防守 / 震荡 / 切换），以及行情的真实质感（健康还是脆弱）。

## 分析步骤

1. **市场分档不只看指数涨跌**：综合指数趋势、市场广度、领涨方向清晰度、情绪温度、流动性 + 政策、盈利预期支持。指数涨但广度差是典型脆弱信号。
2. **指数健康度**：
   - 涨了是真强还是少数权重硬拉？看涨跌分布、创新高 / 新低个数、行业参与度
   - 跌了是全面转弱还是结构调整？看权重与普通股是否同步
3. **行情性质归类**：
   - **全面扩散（健康）**：80% 以上行业上涨、个股普涨
   - **局部抱团（风险）**：少数赛道独涨、其他普跌
   - **弱修复反弹（持续性弱）**：超跌反弹、量能不济
   - **普遍转弱（需防）**：广度恶化、防御板块开始走强
4. **广度恶化是前置信号**：广度恶化常先于指数回落，**指数还在涨但广度先掉头**通常是聚焦过度后的脆弱阶段。
5. **状态切换信号**：
   - 继续成立的条件：什么不变就维持当前判断
   - 切换的前置信号：哪些指标先动会预示切换
   - 判断失效的标志：哪条线被打破就要重写判断
6. **仓位与风格含义**：
   - 进攻档：可加风险敞口、追主线
   - 震荡档：精选个股、控制总仓位、做短不做长
   - 防守档：减仓 + 防守板块 + 现金为王
   - 切换档：减少新进场、等结构清晰

## 建议骨架（不强制）

- 市场档位判断
- 指数与广度对比
- 强弱分布与结构诊断
- 驱动因素与切换信号
- 仓位与风格含义
- 后续观察清单

## 避坑

- 广度弱不一定看空，但要说清风险来自哪里
- 不只看单日异动——要看阶段性广度变化
- 状态尚不清晰时敢说"切换中 / 震荡中"
- 别把"指数没跌"等同于"市场健康"——少数权重撑住的指数最危险
