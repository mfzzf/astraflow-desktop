# 资产配置与组合优化（portfolio-optimization）

> **何时参考**：用户问"资产配置 / 怎么分配仓位比例 / 组合优化 / 再平衡 / 60-40 / 全天候 / 风险平价"。

## 核心目标

**构建多元化投资组合并设计权重分配 + 再平衡规则**。

## 主要框架

### 1. 现代投资组合理论（MPT）
- 在给定风险水平下最大化收益
- 核心：有效边界 + 资本市场线（CML）的约束优化
- 不要直接用原始 MPT——极易产生集中度过高的权重，必须加上下界约束或正则化

### 2. Black-Litterman 模型
- 从市场均衡出发融入主观观点
- 参数 τ（0.025-0.05）和观点不确定度 Ω 平衡市场权重与投资者观点
- 适合"我有特定方向看法"的场景

### 3. 风险预算（Risk Parity）
- 按风险贡献度而非资金比例配置
- 各资产风险贡献不超过总风险的 1/n
- 长期稳定性更优

### 4. 全天候配置
- 风险均匀分布于经济增长上升 / 下降、通胀上升 / 下降四种环境
- 典型配置：股票 30% / 债券 40% / 黄金 15% / 商品 15%

## 优化器选择优先级

1. **等波动加权**（simple baseline）
2. **风险平价**（推荐默认）
3. **均值方差**（需要回报预测，估计精度低）
4. **最大分散化**（低相关追求，高相关环境下作用有限）

## 关键参数

- **lookback window**：60 天为平衡点；< 20 天易过拟合，> 120 天反应迟滞
- **均值方差陷阱**：历史参数估计精度低，样本外夏普比往往只有样本内的 50%——务必加权重约束
- **再平衡触发**：定期（月 / 季）+ 阈值（偏离目标 ±5%-10%）+ 混合方案。中国 A 股需考虑印花税 0.05% + 佣金成本

## 相关性参考

- A 股大小盘相关性 0.85（分散空间小）
- 股债负相关 -0.15-0.20（是配置基础）
- 黄金与权益低相关 0.05
- 加密资产在压力时相关性不稳定

## 约束条件管理

- 无杠杆（权重和 ≤ 1.0）
- 单资产上限：保守 3% / 稳健 5% / 积极 8%
- 行业集中度不超过权益仓位 25%

## 建议骨架（不强制）

- 资产配置建议表（资产类别 / 权重 / 风险贡献率 / 预期收益）
- 优化器配置参数
- 期望风险收益指标（预期收益率 / 波动率 / 夏普 / 最大回撤 / VaR）
- 再平衡规则（频率 / 触发条件 / 估计交易成本）

## 避坑

- 不要直接用原始 MPT——必须加约束
- 均值方差对预期收益估计敏感，错误的预期会显著扭曲组合
- 高相关性环境中最大分散化作用有限——结合宏观情景判断
- 中国市场特色：无资本利得税但有印花税 0.05%、QDII 额度受限、房产占家庭资产 60-70% 需纳入考虑
- 再平衡过频会被交易成本吃掉——成本小于 0.3% 才有意义
