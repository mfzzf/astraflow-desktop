# 资金流与北向（fund-flow）

> **何时参考**：用户问"北向资金今天买了什么 / 外资在买什么 / 资金流向 / 主力流入 / 沪深港通净流入"。

## 核心目标

**判断资金流向是配置信号、风格支持还是短期扰动**——不孤立看单日数据。

## 分析步骤

1. **不孤立看单日**：必须检视连续性、集中度、与指数 / 情绪的共振。**单日数据无法改变配置判断。**
2. **区分四种行为性质**：
   - 趋势性布局：连续多日同向、分布广泛
   - 事件性交易：聚焦特定标的 / 板块、伴随消息
   - 被动配置：指数扩容 / 调样 / ETF 申赎
   - 短期避险：风险事件期间的回流
3. **资金偏好的行业与风格倾向**：是回归核心资产、提升风险偏好，还是防御性配置。
4. **代表性标的与集中度**：高度抱团于权重龙头通常代表共识强化；分散流入则代表整体风险偏好上升。
5. **资金流与价格背离**：流入但股价跌 / 流出但股价涨——往往是反向信号。需要追问哪边更值得相信。
6. **后续可跟踪线索**：通过指数、板块轮动、个股走势验证资金行为是否延续。
7. **区分"单日信号"与"阶段信号"**：阶段性流向才有配置意义——单日流入流出多数是噪音。

## 建议骨架（不强制）

- 30 秒结论（总体流向 / 主要偏好 / 风格变化 / 当前含义）
- 总体流向与指数 / 情绪关系
- 行业 / 风格偏好拆解
- 代表性标的与集中度
- 信号可信度判断与误读风险

## 避坑

- "净流入"不等于利好——可能来自被动调仓 / 指数扩容
- 一天数据无法改变趋势判断，要看连续多日
- 资金偏好不直接等于基本面支持，需结合盈利验证
- 外资偏好在风格切换时反转极快，警惕拐点信号
- 主力资金流（按大单 / 中单 / 小单分）是统计推断，参考意义有限
