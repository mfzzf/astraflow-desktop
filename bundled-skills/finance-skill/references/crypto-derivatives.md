# 加密衍生品（crypto-derivatives）

> **何时参考**：仅在用户**明确要求**加密相关分析时使用——主 SKILL 默认不召唤 crypto。用户问"BTC / ETH / 永续合约 / 资金费率 / 链上数据 / DeFi 收益 / 稳定币流向 / 爆仓热力图"时参考。

## 核心目标

加密衍生品分析框架——**保留方法论，不鼓励高杠杆参与**。

## 主要内容

### 1. 加密期货定价与基差
- 现货-期货基差通常为正（升水 0.5-2%/月）源于资金费率和融资成本
- 基差 = (期货价 - 现货价) / 现货价
- 极端波动（> 3%）意味套利机会
- **交割期货**（季度交割）vs **永续期货**（无到期、通过资金费率平衡价格）

### 2. 资金费率机制
- 永续期货每 8 小时结算一次
- 正费率：long 付款给 short → 期货升水过高（市场看好）
- 负费率：short 付款给 long → 期货贬水过低（市场看空）
- 年化费率 8%-15% 常见
- > 20% 表示泡沫信号
- < 1% 表示低热情

### 3. 爆仓热力图与杠杆分析
- 交易所公布各价位杠杆多空头寸量
- 价位附近杠杆头寸密集 → 潜在爆仓风险大
- 穿过这些价位可能触发止损连锁反应（向下爆空头 → 向上爆多头）

### 4. DeFi 收益分解
- 借贷利率（供应端 APY）由需求与供给决定
- 稳定币借贷 APY 通常 3-8%
- 流动性挖矿 APY = token 奖励 + 交易费收入
- **必须扣除 token 价格风险与无常损失**——名义 APY 高不代表实际收益

### 5. 稳定币流向分析
- USDT / USDC / BUSD 等链上流向（交易所进出、钱包地址浓度）反映市场情绪
- 大量稳定币进交易所 → 做多意愿上升
- 大量稳定币离开交易所 → 资金离场或锁仓 DeFi

### 6. 链上数据指标
- 活跃地址数（市场参与度）
- 交易所流入 / 流出量（鲸鱼动向）
- MVRV 比率（已实现价值与市值比）：≈1 时底部、> 1.2 时顶部（基于历史）

### 7. 套利策略
- 现货-期货基差套利（买现货 + 空期货）
- 跨交易所套利（同资产不同交易所价差）
- 杠杆寿命成本对冲（长期持仓用永续期货融资、费率成本需纳入回报计算）

### 8. 风险与限制
- 加密市场 24/7 交易、流动性可能枯竭（小币种）
- 杠杆风险激增（容易爆仓）
- 监管不确定性（政策冲击可能导致暴跌）
- staking 风险（锁定期内无法提现、slash 风险）

## 建议骨架（不强制）

- 期货基差追踪（现货价 / 期货价 / 基差% / 资金费率 / annualized 成本）
- DeFi 收益估算（借贷 APY / 流动性挖矿 APY / token 价格假设 / 无常损失估计）
- 爆仓风险地图（各价位杠杆头寸密度 / 穿越概率 / 预期连锁反应）
- 链上指标仪表板（活跃地址 / 交易所流向 / MVRV 比 / whale 动向）

## 避坑

- 资金费率持续为正不代表可以无风险套利——实际可能因为现货难获得（个别币种）或期货流动性差而无法执行
- DeFi 收益看起来很高（20%-200%+）——警惕 token 价格崩溃，实际 net 收益常为负
- 爆仓热力图仅是概率信号，不是确定性——市场可以跳过这些价位
- 永续期货杠杆诱人但极度危险——资金费率反转时长头寸会同时被压制
- 链上数据滞后性强（1-2 天）——不应用于日内交易，更适合中周期判断
- **回答用户加密相关问题时务必显示数据和监管风险提示**
