# 政策解读与受益映射（policy-impact）

> **何时参考**：用户问"X 政策出来怎么看 / 监管文件什么意思 / 谁是真正受益方 / 这政策能不能交易"。

## 核心目标

**抓住政策的"增量"价值**——不复述原文，把核心增量、执行强度、受益链条与市场交易阶段讲清。

## 分析步骤

1. **政策"增量"识别**：
   - 首次提出的新口径
   - 强度提升（从鼓励到强制）
   - 支持范围扩大
   - 执行工具明确（财政补贴 / 税收 / 专项贷款 / 采购支持 / 准入放松）
   - 还是旧口径重复？真增量决定交易价值。
2. **政策级别与执行强度**：
   - 顶层文件（国务院 / 中央）：力度大、执行慢
   - 部委政策：相对落地快
   - 地方试点：覆盖面有限
   - 监管指导 / 协会倡议：约束力弱
   配套工具是否明确决定真实力度。
3. **受益链条建立**：
   - 直接受益：政策直接作用的产业 / 企业
   - 间接受益：需求扩散 / 资本开支 / 订单外溢
   - 概念映射：市场愿意交易但业绩关联弱
4. **标的分层**：核心受益龙头 / 容量中军 / 高弹性小票 / 概念映射标的——各自定价逻辑不同。
5. **市场交易阶段**：
   - 政策催化初期：可参与
   - 已提前预期：兑现风险
   - 利好兑现中：边走边看
   - 等待实施细则：观望
6. **政策价值与政策热度可能背离**——脱离市场位置判断，避免追高。
7. **必须讲清"为什么这家公司在这政策下最受益"**——不能把所有相关公司都定义为受益股。

## 建议骨架（不强制）

- 政策核心增量与级别
- 执行工具与强度评估
- 受益链条分层
- 标的受益程度排序
- 市场交易阶段与机会窗口
- 风险与反证因素

## 避坑

- 政策预期常在发布前就被定价——发布后可能无涨幅
- 地方试点政策的全国推广存在不确定性，不能简单放大
- "产业鼓励"与"产业补贴"力度完全不同
- 监管政策同时有受益方和受损方——双向评估
- 别把"新闻当政策"——非正式表述不算政策出台
