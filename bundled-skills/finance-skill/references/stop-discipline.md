# 止损纪律（stop-discipline）

> **何时参考**：用户问"该不该止损 / 止损位怎么定 / 套了怎么办 / 关键位预警 / 触发到目标价该不该卖"。

## 核心目标

**给单笔交易设计三维止损**（价格 + 逻辑 + 时间），并在触发后给明确动作顺序，避免自我合理化。

## 分析步骤

1. **价格止损**：跌破关键支撑（前低 / 5 日线 / 20 日线 / 关键位）或幅度止损 -5% 至 -8%。**确认方式要写清**（收盘破位 / 盘中破位 / 连续破位）——避免被瞬时插针误伤。
2. **逻辑止损**：核心催化失效（利好落空 / 公告澄清）、板块转弱（龙头跌停 / 板块大跌 5% 以上）、龙头负反馈、放量破位（成交量放大 1.5 倍 + 跌破关键位）。
3. **时间止损**：原本预期多快验证，现在拖延的时间成本——横盘或弱势震荡占用资金达到时间阈值（短线 5-10 个交易日、中线 1-2 个月）后降仓或退出。
4. **触发后动作顺序**（这是纪律的核心）：
   - 是先减半再观察、一次性退出、还是盘中打折等待确认
   - 触发后**禁止重新评估持有理由**——评估应在触发前完成
   - 避免"再等等说不定回来"的自我合理化
5. **例外情形**：噪音波动 vs 真正失效要区分——快速插针（盘中跌破后迅速收回）通常是诱空、连续多日趋势性破位才是真失效。
6. **关键位预警与目标价触发**：
   - 关键位触碰 ≠ 突破——需要收盘确认或次日延续
   - 目标价到达不代表必须卖——判断预期兑现度、是否有新催化、若强势穿越则上调目标
   - 目标价触发后回落要重点警惕兑现风险

## 建议骨架（不强制）

- 原始交易计划与失效标准
- 价格 / 逻辑 / 时间三层止损框架
- 例外情形（噪音波动 / 跳空失守 / 快速收回）
- 触发后的明确动作顺序
- 关键位预警与目标价触发处理

## 避坑

- 不能写"感觉走弱"，必须客观化触发条件
- 不能只写"考虑减仓"，要给明确动作
- 把随机扰动当计划终结也是错——区分噪音与真失效
- 触发后**禁止改规则**，避免事后宽松
- 浮盈被打回成本不是止损——是退出原因不同

## 锚定偏差识别（操作类常见陷阱）

用户拿**成本价 / 历史高点 / 跌停板**讨论止损时，第一步先**脱锚**：

- "我成本 35.825 套了 40% 怎么办" → 成本价**不是市场关心的变量**，市场只关心当前价格相对基本面 / 技术位置的偏离
- "回到 X 元我就卖" → 这个 X 是用户的心理锚（成本 / 前高），不是市场决定的卖出位
- "再跌到跌停板我就割" → 跌停板是情绪极端值，不是合理止损位

**正确顺序**：先把讨论锚点从"我亏多少 / 我要回本 / 跌到 X 我才割"切回"X 现在该不该持有"——基于当前的基本面、估值、走势位置重新评估，再进入止损框架（价格 / 逻辑 / 时间三维）。

**典型话术**：
> 先放下成本价。35.825 是你的买入价，不影响 X 现在该不该持有。我们分别看：(1) X 现在的基本面有没有破坏 (2) 走势处于什么位置 (3) 原买入逻辑还在不在。这三个都答完，再决定止损还是继续持。
