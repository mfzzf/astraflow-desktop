# 市场主线与情绪（market-mainline）

> **何时参考**：用户问"今天主线是什么 / 现在市场在交易什么 / 情绪怎么样 / 该不该进场 / 大盘 + 主线判断"。
>
> **输出形式**：本场景结论以 HTML 文件呈现（用 `Write` 工具落盘到当前工作目录，文件名形如 `<topic>-<YYYYMMDD>.html`），对话正文里附 200-300 字摘要 + 文件路径让用户自行打开。

## 核心目标

**识别真主线**——别把一日脉冲误判为主线，别把所有涨停都说成机会。

## 分析步骤

1. **市场环境判断**：强势 / 震荡 / 弱势 / 退潮——看指数强弱、涨跌家数、成交额、情绪股表现，判断当前适合**进攻 / 精选 / 控制 / 观望**。
2. **区分四种热度**：
   - 真主线：连续性强、龙头明确、扩散清晰
   - 次级热点：当日强势但持续性待验证
   - 脉冲题材：一日游、缺乏龙头
   - 跟风分支：主线带动、独立性弱
3. **梯队识别**：龙头 / 中军 / 补涨 / 后排——谁是核心锚点、谁只是跟风、龙头带动性强弱决定题材持续性。**新老龙头切换时分歧最大**。
4. **情绪周期判断**：冰点 / 修复 / 主升 / 高位震荡 / 退潮，结合连板高度、炸板率、高位股反馈、龙头分歧承接。**情绪温度计**：连板高度 + 炸板率 + 涨停数 + 跌停数 + 高位股次日溢价。
5. **主线持续性评估**：产业逻辑扎实度、事件催化连续性、资金合力明确度——分为弱 / 一般 / 较强 / 强。
6. **明日观察重点**：龙头能否弱转强、看分歧后回流、看中军放量新高、看后排掉队。**给 1-3 个最值得观察的核心变量。**

## 建议骨架（不强制）

- 市场环境判断（适合进攻 / 精选 / 控制 / 观望）
- 当前主线识别与催化
- 次级热点与脉冲题材
- 龙头 / 中军 / 补涨梯队
- 情绪周期与风险点
- 明日观察清单

## 避坑

- 不要把所有涨停都说成机会——识别真伪主线
- 情绪周期判断需要证据支撑，不能只用形容词
- 主线不清时敢说"轮动为主、降低预期"
- 一日脉冲 ≠ 主线
- 主线后期的高度题材有兑现风险，不要追高位龙头
