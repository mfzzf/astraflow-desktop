# 管理层体检（management-assessment）

> **何时参考**：用户问"这家公司管理层怎么样 / 老板靠不靠谱 / 治理结构如何 / 能不能信任公司说的话"。

## 核心目标

**判断管理层是否值得信任，以及这种信任应建立在什么边界上。**事实和判断分开写，不做人格化评价。

## 分析步骤

1. **评估范围**：创始人 / 董事长 / CEO / CFO / 控股股东 / 董事会结构。控制权高度集中时，把控制结构本身纳入评估。
2. **背景与履历**：行业和经营经验是否与当前职责匹配、核心团队是否稳定、历史职业轨迹是否有加分或减分项、是否存在频繁更换 / 高层动荡。
3. **激励一致性**：持股比例与近年增减持行为、薪酬与激励结构、业绩承诺或考核目标是否合理、激励是否鼓励短期粉饰还是长期建设。
4. **执行与资本配置历史记录**：过去几年关键经营兑现情况、并购 / 回购 / 分红 / 扩产等决策质量、面对逆风期的反应能力、对外沟通是否前后一致。**历史承诺多次落空显著降低信任等级。**
5. **治理结构与红旗信号**：关联交易与利益输送风险、财务透明度、重大事项披露及时性、控股权结构复杂性、是否存在监督不足。识别"需要更谨慎"的地方而非定罪。
6. **输出信用判断与跟踪重点**：是否可给予高信用、信用建立在什么前提、后续最应跟踪哪些管理与治理信号。

## 建议骨架（不强制）

- 信用等级（高 / 中 / 低）+ 最大担忧点
- 关键人物与治理结构
- 激励与利益一致性
- 执行与资本配置观察
- 治理风险与红旗信号（3-5 条）
- 后续跟踪重点

## 避坑

- 事实与判断分开写，避免把印象当证据
- 不做人格化评价，落在胜任力 / 激励 / 治理结构
- 既要看"会不会做事"，也要看"会不会为股东做事"
- 红旗信号要明确，但不能用单一传闻直接定性
