# 外汇与大宗商品（forex-commodity）

> **何时参考**：用户问"美元兑人民币 / 日元 / 黄金 / 原油 / 铜 / 大豆 / 套息交易 / 商品期货 / 汇率对冲"。

## 核心目标

**评估外汇 / 商品趋势 + 跨境投资组合 + 汇率对冲**。

## 主要内容

### 1. 外汇基本面分析
- **购买力平价 PPP**：长期汇率 = 本币 / 外币 × 相对物价水平
- **利率平价 IRP**：远期升贴水 = 前期汇率 × (1+r_domestic)/(1+r_foreign)
- **国际收支账户**：经常账户赤字通常导致贬值
- **央行政策**：利率 / 量宽 / 外汇干预

### 2. FX 期权波动率
- FX 市场常用 delta 空间报价而非 strike 空间
- 25-delta 风险反转（RR = put IV - call IV）反映斜度偏好
- 蝶式（butterfly）反映微笑程度
- 短期（1 个月）波动率普遍 < 中期（3-6 个月）（正常 term structure）

### 3. 商品周期结构
- 套利成本（storage + convenience yield）导致期货升贴水或反向市场
- **顺利市（contango）**：近月 < 远月，出现在供给充裕期
- **反向市（backwardation）**：近月 > 远月，出现在供给紧张期
- 反演速度（速度快 → 市场预期逆转）是套利信号

### 4. 商品定价驱动
- 供需平衡：库存 - 产量数据领先价格
- 汇率：美元升值压低商品价格
- 实际利率：负实际利率提升商品需求
- 情绪与 VIX：风险厌恶时商品 as-risk-asset 受压

### 5. 套息交易框架
- 高息货币（澳元 3-3.5% / 新西兰元 5%+）vs 低息货币（日元 0% / 瑞郎 1.5%）
- 净利差通常在 2-3%
- 需考虑汇率波动（typical volatility 8-12%）
- 持有期 2-3 年时年化回报可达 2-4%（如不大幅贬值）

### 6. 商品投资方式对比
- 现货：市场小、流动性受限
- 期货：流动性好但需对冲展期风险
- ETF：便利但可能存在跟踪偏差
- 基金：分散化但成本相对高
- 中国商品期货需注意交割月前的展期成本

### 7. 对冲策略设计
- 经营企业：远期 / 期货锁定采购 / 销售成本（完全对冲）
- 投资组合：套息交易平滑外币汇率风险（部分对冲）
- 宏观基金：VIX / 波动率指数做商品风险管理（动态对冲）

### 8. 市场信号与时机
- CFTC 持仓报告（非商业 vs 商业持仓比例反映投机情绪）
- 期现差（升水 > 0.5% = 期货贵，做空机会）
- 关键库存水位：原油 OPEC 库存目标、铜 LME 库存、农产品 USDA 数据

## 建议骨架（不强制）

- 汇率基本面评估（PPP / IRP / 实际现货 / 远期升贴水 / 政策倾向）
- 商品供需平衡（产量 / 消费 / 库存变化）
- FX 期权波动率曲面（ATM vol / 25-delta RR / butterfly）
- 套息交易回报计算（年化利差 - 预期汇率波动 + 滚动成本）

## 避坑

- PPP 长期成立（3-5 年+）——但短期汇率完全由资本流动驱动，PPP 短期预测无用
- 套息交易遇风险事件（金融危机 / 地缘冲突）所有高息货币同时贬值——无法对冲
- 商品期货展期成本在 backwardation 下侵蚀收益，contango 下逐月展期持续亏损
- 不要用商品作为通胀对冲——大部分时候商品与通胀相关性不稳定（与实际利率相关性更强）
- FX 波动率微笑在极端环境下陡峭——OTM 期权远超 BS 理论值，警惕 model risk
