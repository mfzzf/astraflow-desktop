# 涨停龙头博弈与龙虎榜（leader-game）

> **何时参考**：用户问"今天龙头是哪只 / 涨停结构怎么看 / 这只票龙虎榜上来了 / 高位股能不能接力 / 连板梯队"。

## 核心目标

**分析短线涨停结构与龙头博弈**——不简单"涨停多就强"，要看高位反馈与次日溢价。

## 分析步骤

1. **涨停结构判断**：涨停数量、跌停数量、炸板率、连板高度、空间板（最高板）、高位股反馈、昨日连板溢价 → 情绪修复 / 加强 / 局部退潮 / 全面退潮。
2. **连板梯队完整度**：谁是空间板（最高）、二板、三板等关键位、梯队是否完整、是否存在断层、是否有新老龙头切换迹象。**断层是退潮信号**。
3. **题材分类与强度**：
   - 主流题材：涨停数多 + 连板高 + 有中军
   - 强支线：龙头明显但容量小
   - 日内脉冲：缺乏续航
   - 跟风噪音：纯情绪 / 板块带动
   不能平等看待。
4. **龙头博弈结构**：绝对龙头 / 潜在卡位股 / 最强补涨 / 高位风险股——地位来自先手 / 辨识度 / 板块带动 / 换手质量还是纯情绪。
5. **龙虎榜席位属性**：
   - 知名游资：风格连续性强、能带动跟风
   - 趋势资金：买完不一定走
   - 量化席位：高频换手、信号意义弱
   - 真机构 / 假机构：要区分（基金专用席位 vs 普通机构席位）
   - 对倒：异常对敲信号
   **不能见游资就看多、见机构就看好——理解历史风格是关键**。
6. **明日博弈焦点**：龙头分歧转一致还是高低切、强分支回流还是新题材试错、连板继续还是趋势承接——给 1-3 个最值得观察的核心变量。

## 建议骨架（不强制）

- 涨停结构与情绪判断
- 连板梯队与完整度
- 题材强度与分类
- 核心龙头 / 卡位股识别
- 龙虎榜席位属性与买卖力量对比
- 明日博弈焦点与风险提示

## 避坑

- 不能"涨停多就强"——看高位反馈与次日溢价
- 游资接力一致性太强 = 兑现风险大（次日开盘高度压制）
- 假机构席位容易误导——龙虎榜本身只代表波动不代表趋势
- 高位龙头追涨的赔率最差
- 龙虎榜仅支持 A 股龙虎榜数据，港股 / 美股没有等价机制
