# 宏观行业个股传导（macro-transmission）

> **何时参考**：用户问"加息对 X 影响 / GDP 数据出来对市场怎么看 / 美元升值受益谁 / 政策对哪个行业有利 / CPI 高了配什么"。

## 核心目标

**建立宏观变量 → 行业盈利 → 市场风格的完整传导链条**，不允许"所有行业都受益"的泛化结论。

## 分析步骤

1. **定义宏观变量本质**（不停留新闻表面）：利率上下行、汇率波动、信贷扩缩、财政发力、地产政策、全球需求、通胀压力——提炼对资产定价最重要的核心变量。
2. **拆解传导路径，必须经过中间变量**：
   - 利率变化 → 估值中枢 / 融资成本 / 风险偏好
   - 汇率 → 出口竞争力 / 原材料成本
   - 财政 → 基建投资 / 设备需求
   - 货币宽松 → 流动性 → 风险资产
3. **区分一阶 / 二阶受益**：
   - 一阶：最直接（航运、外贸制造受汇率直接影响）
   - 二阶：滞后但弹性更大（设备、零部件、材料）
4. **个股映射分层**：
   - 核心受益股：逻辑最直接、确定性高
   - 中军型大市值受益股：稳定增长
   - 高弹性小票：弹性大、波动也大
   - 概念相关弱映射：仅市场愿意交易
5. **传导时滞与验证指标**：哪些行业最先反应在股价、哪些要等数据验证、哪些在业绩端更晚体现（通常 1-2 个季度）。
6. **判断适合"交易性主题"还是"中期配置"**：先反应的方向适合短线交易，业绩验证型适合中期持仓。
7. **必须写出清晰传导链**：A → B → C → 受益方，不允许跳过中间步骤。

## 建议骨架（不强制）

- 宏观变量核心变化
- 传导路径与中间变量
- 一阶 / 二阶受益行业
- 代表性个股与受益原因
- 传导时滞与关键验证指标
- 投资机会与风险

## 避坑

- 经济数据与股价表现时常背离——传导不是直线
- 同一宏观变量在不同历史阶段传导路径可能反向
- 市场往往提前反应——数据公布时可能已充分定价 / 反向交易
- 政策传导存在"时滞"和"失效"风险——不假设政策意图必然兑现
- 美联储 / 中国央行决议要点出本地时间 + 北京时间双标
