# 个股初探（stock-first-look）

> **何时参考本文件**：用户第一次接触一只陌生股票（"这家公司怎么样 / 看看 XX / 分析下 YY"），需要在短时间内建立"够用且可讨论"的第一层认知；或者看到一只热门股但不熟悉，要快速补课。

## 核心目标

不是全面尽调，而是回答五件事：**做什么 / 为什么值得看 / 市场在争什么 / 大致贵不贵 / 继续研究前先补哪几块**。

## 分析步骤

1. **确认对象与上下文**：股票名称、代码、上市地、所属行业；用户目的是初次研究、判断深入价值，还是临时补课/口头汇报。
2. **基础事实压缩**：公司卖什么 → 主要客户 → 收入利润主要来自哪些业务模块 → 产业链位置 → 近阶段最重要的经营或市场变化。**不**追求历史沿革。
3. **为什么现在值得看**：催化（财报/公告/政策/产品/订单/资本运作）、行业景气与板块轮动、业绩-估值-预期之间的预期差、价格位置或讨论度突变。**没有强催化**也要明说"目前观察名单而非优先研究对象"。
4. **关键变量与财务快照**：3-5 个最重要经营变量 + 收入/利润/利润率/现金流/增长结构基础指标；对估值最敏感的单一变量优先点出。
5. **估值位置与分歧初判**：明显低估 / 合理 / 偏高，市场预期保守 / 均衡 / 过热；估值弹性主要来自业绩兑现、逻辑重估，还是情绪扩散。**列最重要的看多与看空各 2-3 条**，避免单边叙事。
6. **是否继续研究的落地建议**：继续 / 暂缓 / 仅观察；若继续，下一步该补哪几个具体问题；若暂缓，原因是信息不足、逻辑不清还是赔率不佳。

## 热门股变体

如果是"已经火的股票要快速看懂"，多一步：先确认**热度来源**（突发事件 / 板块映射 / 连续涨停放量 / 媒体集中讨论），不同热度来源决定后续解读框架。

## 建议骨架（不强制）

- 30 秒结论：做什么 / 为什么值得看 / 当前最大争议 / 是否继续研究
- 公司基础认知（主营、客户、产业链位置、赚钱方式）
- 当前最重要 3 个关注点
- 关键经营与财务快照（带数字）
- 估值位置与预期差判断
- 多空要点
- 下一步研究清单

## 避坑

- 数据缺失不要用"较高/较好/不错"模糊掉，**显式标"该数据待补"**
- 没有强催化时不要硬挤一个理由出来，**直接说观察而非优先**
- 不展示中间检索路径或思考草稿
- 港股 / 美股代码、ADR 与本地股、同名公司**先核对身份**再说估值
