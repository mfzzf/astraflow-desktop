# 放量异动与跳空归因（abnormal-detection）

> **何时参考**：用户问"X 今天为什么大涨 / 放量是什么意思 / 这只票跳空怎么解读 / 盘中突然异动 / 主力出货还是吃货"。

## 核心目标

**快速识别放量 / 跳空驱动来源**，区分信息含量与情绪扰动，判断可信度与后续跟踪价值。

## 分析步骤

1. **确认偏离常态**：成交额 / 成交量是否显著抬升 vs 近阶段常态水平、换手率同步明显变化、放量是持续型还是脉冲型、主要集中在单一时段（开盘 / 尾盘 / 中段）。
2. **驱动排查四类**：
   - **事件驱动**：公告 / 新闻 / 政策 / 财报 / 产品 / 订单
   - **板块驱动**：同行集体异动 / 题材发酵 / 龙头带动
   - **技术驱动**：关键位突破 / 跌破 / 止损踩踏
   - **资金驱动**：情绪抢筹 / 游资接力 / 博弈换手
3. **放量质量判断**：
   - 价格是否顺放量方向推进（放量 + 大涨 vs 放量 + 滞涨）
   - "只放量不走价"通常是分歧信号
   - 放量后是否带来结构重估还是一天脉冲
   - 当前市场是否支持延续
4. **位置决定含义**：
   - 低位放量：吸筹 / 启动信号
   - 高位放量：可能出货
   - 突破放量：有效突破信号
   - 冲高回落放量：诱多 / 兑现
   - 下跌放量：恐慌 / 趋势加速
5. **跳空归因**：
   - 事件驱动型（扎实催化）：跳空可持续
   - 情绪强化型（市场过度反应）：跳空易回补
   - 补涨补跌型（相对强弱调整）：跳空有结构意义
   - 技术位置触发型：止损踩踏 / 多头止盈
6. **后续验证重点**：次日是否承接、关键位置是否站稳、题材热度是否继续支持、成交回落后价格是否维持强势。

## 建议骨架（不强制）

- 放量概况（时段 / 方向 / 成交变化具体数字）
- 归因分类四维评分（事件 / 板块 / 技术 / 资金）
- 放量质量判断（可信度 / 支持证据 / 警惕风险）
- 后续跟踪重点与验证条件
- 跳空性质判断

## 避坑

- 不能把所有放量都等同于利好——高位放量往往是出货信号
- 成交量极限扩大可能反映流动性枯竭而非资金涌入
- 事件催化常在发布前已被市场提前反应——公告出现时反而是利空兑现
- 跳空缺口易被高估——历史数据显示大部分缺口在数周内被回补
- 盘中瞬时放量可能只是程序交易 / 期权对冲——与基本面无关

## 可执行工具

**数据 skill 边界**：westock 提供实时分时 / 资金流（`westock-data minute`、`asfund`），新闻 / 公告由 `wenda_news_query` 等覆盖；**但"放量阈值判定 + 量价配合度评分 + 跳空缺口分类"是结构性算法判断**，数据 skill 不给你结论。

- 涉及量价配合度算法判断 → 用 `scripts/price-action/basic_indicators.py` 的量比 / OBV / 量价背离检测
- 涉及跳空缺口对应的 K 线形态 → 用 `scripts/price-action/candlestick_patterns.py`

调用约定见 `price-action-tools.md` 末尾的"可执行工具"section。
