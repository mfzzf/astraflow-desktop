# 题材周期与龙头（theme-lifecycle）

> **何时参考**：用户问"X 题材还能不能玩 / 现在题材到哪一阶段 / 龙头是哪只 / 这个题材会不会扩散 / 是题材初期还是末期"。

## 核心目标

**对题材所处生命周期阶段做前瞻判断**，不做"事后诸葛亮"。

## 分析步骤

1. **题材起点**：政策 / 产业趋势 / 业绩 / 技术突破 / 海外映射 / 市场自发演绎。**讲清楚市场为什么关注这个题材**。
2. **阶段判断**：
   - 预热期：少数股先动，板块未形成
   - 启动期：龙头出现，涨停扩散
   - 发酵期：连板上升，中军跟随放量
   - 高潮期：连板高度极致，跟风普涨
   - 分歧期：龙头分歧、补涨股开始掉队
   - 退潮期：龙头跌停、连板减少、跟风全跌
   **判断必须给依据**（涨停扩散、龙头高度、成交额、后排反馈、负反馈率）。
3. **结构分析**：
   - 情绪龙头：辨识度最高、纯情绪驱动
   - 容量中军：稳定上涨、机构参与
   - 弹性补涨：位置低、跟随龙头
   - 后排噪音：纯情绪 / 跟风
   地位是否稳固、是否出现掉队信号。
4. **资金参与强度**：纯游资题材持续性通常弱（3-5 天）；需要产业逻辑 + 机构中军合作才能持续强（数周到数月）。
5. **持续性评估**：逻辑扎实度 / 催化连续性 / 板块扩散能力 / 龙头能否持续打开空间 / 中军能否承接增量资金 → 弱 / 一般 / 较强 / 强。
6. **交易策略与阶段匹配**：
   - 启动期：追龙头
   - 发酵期：分歧低吸 + 加中军
   - 高潮期：只看龙头 / 减仓
   - 分歧期：观望 / 看新一轮一致性
   - 退潮期：回避 / 看调整后能否反包

## 建议骨架（不强制）

- 题材驱动与起点
- 当前阶段与判断依据
- 标的结构与地位分层
- 资金参与与强度
- 持续性评级与风险点
- 交易策略建议（与阶段匹配）

## 避坑

- 不能事后诸葛亮——要对未来趋势做前瞻判断
- 题材退潮时才看到"原来不值得做"没意义——要在中期识别降速信号
- 龙头切换通常先从"新龙头与老龙头分歧加大"开始
- 高潮期追龙头的胜率最低 / 风险最高
