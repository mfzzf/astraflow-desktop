# 护城河与公司质地（moat-quality）

> **何时参考**：用户问"这家公司有没有护城河 / 壁垒 / 长期竞争优势 / 是不是好公司 / 长线票"，或要做长期持有判断、优质公司初筛。

## 核心目标

**区分"短期领先"和"结构性优势"**——好公司不等于好股票，但没有护城河的"高景气标的"长期配不上高估值。

## 分析步骤

1. **明确要验证的护城河命题**：成本优势是否足够压制同行 / 品牌 + 渠道 + 客户关系是否形成进入门槛 / 技术 + 专利 + 标准 + 生态是否构成锁定 / 网络效应 + 规模效应 + 牌照是否具备持续性。把"我说它有护城河"具体化成可验证的命题。
2. **优势作用机制**：是帮助获取客户、维持价格、提高利润率，还是稳住份额；影响的是收入端、成本端还是回报端；行业共性还是公司特有。
3. **可验证证据（这是核心）**：市场份额与变化趋势、毛利率 / 费用率 / ROE / ROIC 长期表现（多年高且稳定）、客户粘性 / 复购率 / 切换成本、行业内对手结构与进入难度。**没有证据的护城河不算护城河。**
4. **持续性反向审视**：技术替代与商业模式替代风险、监管政策变化、上下游议价结构改变、新进入者 / 跨界者 / 价格战冲击。只有经得起反向审视的优势才值得定为长期护城河。
5. **是否能转化为股东回报**：长期 ROE / ROIC 高、管理层能否把优势转化为合理资本配置、当前估值是否已过度透支这份优势。
6. **公司质地分项打分（不强制几分制）**：业务质量 / 增长质量 / 现金流质量 / 治理质量 / 资本配置质量——任何一项明显短板都要点出。

## 建议骨架（不强制）

- 护城河结论摘要（强 / 中 / 弱 / 暂不成立 + 核心优势来源 + 最大削弱风险）
- 护城河命题
- 证据矩阵（优势类型 / 证据 / 当前判断）
- 持续性分析（强化因素 / 削弱因素）
- 对回报的意义
- 后续跟踪信号

## 避坑

- 必须落到可验证证据，不停留在口号
- 区分阶段景气与结构性壁垒——阶段景气**不是**护城河
- 同时写增强与削弱因素，不单边美化
- 证据不足时直接说"尚不能证明护城河成立"
- 别把"知名品牌"当护城河，要看品牌**是否带来定价权**
