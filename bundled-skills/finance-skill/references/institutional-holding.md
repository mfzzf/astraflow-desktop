# 机构持仓与拥挤度（institutional-holding）

> **何时参考**：用户问"机构在买什么 / 这只票拥挤吗 / 抱团股该不该追 / 基金重仓股 / 13F / 公募调仓"。

## 核心目标

**判断是"优质共识资产"还是"脆弱拥挤交易"**——拥挤本身不必然是风险，但要看是怎么被堆出来的。

## 分析步骤

1. **拥挤度多维判断**：基金持仓比例、持仓集中度、同类机构共识程度、股价位置、估值水平五维度综合。
2. **拥挤来源分类**：真正优质基本面 / 赛道高景气 / 流动性偏好 / 机构缺乏替代品 / 业绩确定性溢价——区分"好公司自然拥挤"和"资金抱团推高"。
3. **拥挤的双面性**：
   - 业绩持续兑现期可维持很久
   - 景气 / 风格变化时去拥挤剧烈（双杀：估值压缩 + 抱团瓦解）
4. **当前预期一致性**：若市场对公司主要预期过于统一，**边际上更容易出现"超预期也不涨"**。
5. **去拥挤触发器**：业绩低于高预期、增速放缓、行业景气转弱、风格切换、监管 / 估值压缩。
6. **机构增减持解读**：
   - 机构增持 ≠ 一定利好——可能只是被动加仓 / 季节性
   - 长线机构新进 = 中期认可
   - 短期高换手资金进出 = 交易信号
   - 减持要分清是兑现还是轮换
7. **策略建议分层**：可继续跟随的共识资产 / 只适合回调后参与 / 需重点防去拥挤风险。

## 建议骨架（不强制）

- 拥挤度等级（低 / 中 / 高 / 过度）
- 拥挤来源分析
- 拥挤的正反面权衡
- 当前一致性预期与风险等级
- 去拥挤触发因素清单
- 策略分层

## 避坑

- 不把拥挤简单等于风险——高质量共识与脆弱共识差异巨大
- 机构减持可能是兑现也可能是轮换，不机械解读
- 估值 / 位置 / 预期三维度缺一不可
- 小众垄断的"非共识"持仓有时风险更大——市场关注度低、难以快速纠正
- 公募季报披露有滞后（披露时持仓可能已变化）
