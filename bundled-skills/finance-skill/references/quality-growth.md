# 质量增长匹配（quality-growth）

> **何时参考**：用户问"高增长能不能持续 / 是不是真增长 / 高质量复利 / 增长有没有水分 / 股息会不会持续涨"，需要判断增长是否真正有质量、是否值得付出更高估值。

## 核心目标

**区分"暂时很快"和"真正有持续性"**——高增长≠高质量，必须拆解来源与代价。

> **取数提示**：判断增长质量要看 ROE / 利润率 / 现金流的长期趋势——westock-data 等官方源默认只返回约 3 年，3 年看不出"是否一直高质量"还是"刚转好"。取数主动写"近 10 年"（如"X 近 10 年营收净利 ROE 毛利率 经营现金流"）。

## 分析步骤

1. **确认增长命题**：是收入高增、利润高增、分部结构性高增、还是困境后恢复性增长。不同类型标准不同。
2. **拆解增长来源**：量增还是价增 / 老产品扩张还是新品 / 内生还是并购并表 / 单一大客户还是多客户普遍 / 景气顺风还是份额提升。**来源决定纯度。**
3. **盈利与现金质量同步检验**：利润率是否同步改善、费用投放是否失控、现金回款是否跟得上、库存 / 应收 / 资本开支是否异常拉升。增长依赖激进让利 / 压货 / 透支现金的，**显著降级**。
4. **判断持续性**：当前驱动能维持多久、未来一年是否仍有同样强的催化、是否存在高基数 / 需求前置 / 补贴退坡、供给变化会否削弱质量。
5. **最可能的失速路径**：单一产品周期见顶 / 大客户回落 / 竞争加剧导致价格回吐 / 景气拐点 / 政策变化。**不写泛风险，要点出最容易爆雷的点。**
6. **资本回报质量**：长期 ROE、ROIC 水平（消费龙头 ROE 常年 >20%、ROIC > WACC 5pp 以上为优秀）、高回报来源是真实经营优势还是财务杠杆、收益质量是否有大量一次性扰动。
7. **再投资空间与分红增长（如果是高股息成长）**：主业扩张空间、新产品 / 新区域 / 新客户能否延续增长、若空间收窄未来回报是否转向分红 / 回购、分红增长是否与盈利现金流匹配。

## 建议骨架（不强制）

- 增长质量等级（高 / 中 / 低）
- 增长来源拆解（量 / 价、内生 / 外延、产品 / 客户、景气 / 份额）
- 盈利与现金含量分析
- 持续性支撑点 + 约束点
- 资本回报质量（ROE / ROIC、护城河支撑）
- 再投资空间与分红增长逻辑
- 最可能的失速路径（3 条）
- 跟踪指标与验证周期

## 避坑

- 高增长≠高质量，拆解来源与代价
- 不能只看收入增速，利润和现金要同步看
- 区分一次性拉动与可持续增长，不线性外推
- 高基数 / 补贴 / 渠道压货支撑的增长要打折
