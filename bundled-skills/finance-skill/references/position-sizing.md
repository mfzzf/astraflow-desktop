# 仓位决策与加减仓（position-sizing）

> **何时参考**：用户问"该买多少 / 加仓还是减仓 / 重仓还是轻仓 / 仓位怎么分 / 浮盈了能不能再加"。

## 核心目标

**仓位由可承受风险反推，不由主观信心正推。**信心高 ≠ 应该重仓。

## 分析步骤

1. **风险预算口径**：单笔最大可承受亏损（账户的多少 % 算上限）、整体仓位天花板、行业 / 风格集中度上限。先定边界再谈仓位。
2. **不确定性决定首笔大小**：不确定性越高，首笔越克制（10-20% 的试错仓为常见起点）；后续验证再放大。
3. **波动与止损距离决定仓位**：止损距离过远应**缩小仓位**而不是放宽纪律。粗略：单笔风险预算 ÷ 止损距离 = 可承受仓位。
4. **组合相关性检查**：新交易是否导致板块 / 风格过度集中、是否与现有持仓高度同涨同跌。已高配某板块时新仓应按"补充暴露"而非独立头寸处理。
5. **加仓 / 减仓的触发条件**：
   - 加仓：原逻辑被强化、走势符合预期、量价配合、环境支持升级。**不在已涨幅的浮盈基础上盲目加码。**
   - 减仓：原逻辑被削弱、走势偏离预期、出现替代标的、环境恶化、达到目标止盈位
   - 分批止盈：先回收成本（卖 1/3）、再吃主升、留底仓做潜在第二段
6. **压力测试仓位**：按真实波动想一遍——开盘跳空、连续不利波动、环境突变是否还能接受。如果不能，**当前仓位偏大**。

## 建议骨架（不强制）

- 风险预算与仓位口径
- 单笔止损距离与波动评估
- 组合相关性与现有暴露
- 建议首笔仓位与扩仓条件
- 加减仓触发清单
- 压力测试与可执行性确认

## 避坑

- 仓位是风险决策不是方向决策——不要把仓位决策与个股选择混淆
- 信心高不等于该重仓，浮盈不等于无限加码
- 没有止损位就没有仓位计算，止损距离是仓位的分母
- 组合相关性是隐藏风险——3 只半导体加在一起就是一笔半导体仓位
