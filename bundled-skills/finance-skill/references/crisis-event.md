# 危机/反转/周期拐点（crisis-event）

> **何时参考**：用户问"周期股拐点到了吗 / 反转逻辑 / 困境反转 / 景气周期判断 / 地缘风险 / 危机后还能买吗"。

## 核心目标

**判断周期景气位置 + 反转逻辑兑现路径**——拐点初期 / 中段 / 高位的策略完全不同。

> **取数提示**：判断周期位置必须看完整周期（往往 5-10 年才走完一轮），westock-data 等官方源默认只返回约 3 年、跨不过一个周期。取数主动写"近 10 年"（如"X 近 10 年营收净利 毛利率 ROE 历年"），才能看出当前处于周期的哪个阶段、对比历史底部/顶部。

## 分析步骤

1. **周期景气位置判断**：综合价格、库存、产能利用率、开工率、盈利趋势：
   - 下行尾声：价格触底、库存见底、产能开始出清
   - 底部磨底：等待催化
   - 上行初期：价格反弹、库存回升信号
   - 上行中段：量价齐升、利润持续扩张
   - 高位见顶：库存堆积、价格见顶但未确认
   - 下行初期：开工率开始下滑
   **核心不是看股价表现，而是看供需与产业链库存的实际变化**。
2. **供需拆解**：
   - 需求端：宏观增速 / 政策刺激 / 行业新增产能需求
   - 供给端：新产能投放时间 / 旧产能出清速度 / 进口替代
   - 库存链条：原料 → 中间品 → 成品的库存周期演变
   - 价格变化通常领先库存 1-3 个月，库存变化领先开工率 1-2 个月
3. **领先 / 同步 / 落后指标区分**：
   - **领先**（提前 2-4 周）：原油价格 / 铜价 / 大宗期货 / 高炉开工率 / 水泥产量
   - **同步**（滞后 1 个月）：工业企业利润 / 产能利用率 / PMI
   - **落后**（滞后 3-6 个月）：产能投资 / 企业扩产计划
4. **拐点信号质量**：
   - 短期交易性反弹（库存补充期脉冲，2-4 周）
   - 中周期上行拐点（产能出清完成、新一轮需求周期开启，6-24 个月）
   - 区分关键："是否有基本面支撑的新驱动"而非单纯价格反弹
5. **个股弹性与风险等级**：
   - **高弹性标的**：低成本企业，上行期超涨、下行期深跌（Beta > 1.5）
   - **成本优势龙头**：下行期仍有盈利、上行期相对稳定
   - **资源禀赋型**：盈利与大宗价格直接挂钩
   - **纯交易映射**：题材股、无基本面驱动
6. **拐点初中后期策略差异**：
   - **初期**（确认触底 + 库存低位 + 价格开始反弹）：高弹性更优（胜率最高）
   - **中段**（价格 + 利润上升、库存温和上升）：龙头最优（确定性最高）
   - **高位**（库存堆积 + 价格横盘 + 利润预期下修）：逐步减仓、防守为主
7. **地缘政治冲击**：供给突然中断（俄罗斯限制粮食 / 能源出口）导致期货跳涨，但若需求无法立即扩张则最终回落。
8. **反转逻辑五问**：
   - 困境来源是什么？
   - 已经出现哪些改善证据？
   - 改善是短期脉冲还是趋势反转？
   - 还需多久兑现、资源是否充足？
   - 失败边界在哪里（单一变量破坏逻辑的阈值）

## 建议骨架（不强制）

- 景气位置判断（当前阶段 + 依据）
- 供需拆解与库存链条
- 领先指标监控清单
- 个股弹性与风险分类
- 拐点初中后期策略建议
- 下行风险与反转失败边界

## 避坑

- 不要把一次价格反弹当拐点——关键是供给侧是否真的出现瓶颈（产能出清 / 环保限产）
- 高弹性标的在下行期非常惨烈——拐点不确定时不要进
- 政策刺激有滞后性（宣布 → 落地 → 显现需要 3-6 个月）——不追风口
- 库存周期长度不确定——有时 1.5-2 年完成一个周期，有时拉长到 3-4 年
- 地缘政治冲击常导致供给侧高波动——初期看起来像拐点但实际是阶段性价格冲击
