# 事件驱动短线催化（event-catalyst）

> **何时参考**：用户问"X 利好 / 政策来了怎么交易 / 这个催化能持续多久 / 短线机会 / 题材发酵后还能追吗"，需要在 3-10 个交易日窗口判断催化质量与博弈结构。

## 核心目标

**站在交易视角**讲清"催化、预期、博弈、路径"，不是只贴利好/利空标签。

## 分析步骤

1. **界定事件性质**：政策驱动 / 产业催化 / 公司公告 / 业绩预告 / 海外映射 / 市场传闻 / 板块联动——不同类型交易持续性完全不同。
2. **判断催化级别（强 / 中 / 弱 / 噪音）**：
   - 是否真实明确（落地条件清晰 vs 模糊预期）
   - 是否足够"新"（首次提出 vs 老调重弹）
   - 是否容易被市场理解（直观 vs 需教育）
   - 是否能外溢到板块（产业链可映射 vs 单点利好）
   - 是否有持续跟踪价值（多期催化 vs 一次性）
3. **市场预期差（短线机会的核心）**：市场此前是否已有预期、当前分歧在哪里、哪些资金会先反应、预期差是否足够形成价格弹性。**预期已充分定价的"利好"反而是兑现风险。**
4. **识别交易结构**：适合追涨 / 分歧低吸 / 等回调 / 等确认 / 不参与，结合个股地位、板块强度、筹码位置、市场环境给交易实战判断。
5. **情景推演**：
   - 最强：催化被持续强化，股价 / 板块如何演化
   - 中性：有冲高但持续性一般
   - 弱势：高开低走或冲高回落说明什么
6. **核心观察变量（决定催化能否转化为行情）**：龙头是否继续晋级、板块是否扩散、中军是否放量承接、跟风股是否大面积掉队、次日溢价是否超预期。

## 建议骨架（不强制）

- 短线交易背景
- 核心催化说明（强 / 中 / 弱 / 噪音）
- 市场预期与博弈点
- 多空情景推演
- 关注变量与观察重点
- 交易结论（可跟踪 / 可博弈 / 需谨慎 / 建议回避）

## 避坑

- 必须站在交易视角，不只说利好 / 利空
- 不把单一新闻强行上升为长期逻辑
- 事件强度不足时直接说"降低预期"
- 催化预期已充分定价时的"利好兑现"是常见陷阱——"buy the rumor, sell the news"
- 政策催化的产业链映射要真——名义相关但实际受益弱的标的别推
