# 固定收益与可转债（fixed-income）

> **何时参考**：用户问"债券 / 国债 / 城投债 / 信用债 / 久期 / 凸性 / YTM / 可转债 / 信用利差 / 国债期货"。

## 核心目标

**评估固定收益资产 + 信用风险 + 利差变化**。

## 主要内容

### 1. 债券定价核心
- 现金流折现法：附息债 = Σ票息/(1+y)^t + 面值/(1+y)^n
- 到期收益率 YTM 是内部收益率，与市价反向关系
- **修正久期 MD = -dP/P ÷ dy**：量化利率 1% 变化对价格的影响
- DV01 = MD × P × 0.0001

### 2. 久期体系应用
- **Macaulay 久期**：现金流时间加权
- **修正久期**：风险管理用
- **有效久期**：含权债（可赎回债 / MBS）
- **关键利率久期 KRD**：分解利率曲线各期限风险

### 3. 凸性修正
- 价格变化 ≈ -MD·Δy + 0.5·CX·(Δy)²
- 凸性为正：利率下行涨幅超预期，利率上行跌幅小于预期
- MBS 等可赎回债凸性为负（提前偿还上限）——特别注意

### 4. 信用分析框架
- **Altman Z-Score**：Z = 1.2×流动性 + 1.4×盈利积累 + 3.3×盈利能力 + 0.6×杠杆 + 1.0×周转率
  - Z > 2.99：安全
  - 1.81-2.99：灰色
  - < 1.81：危险
- **Merton 模型**：将股权视为资产的看涨期权，反推资产价值与违约概率

### 5. 中国特色市场
- 国债 / 政金债（极低信用风险）→ 城投债（隐性政府背书但有分化）→ 信用债 / ABS（需深入评估）
- 银行间市场（90% 交易量、OTC 询价）vs 交易所市场（10%、集中撮合）
- **城投四维评估**：区域财政实力 40% + 平台层级 25% + 平台地位 20% + 债务结构 15%

### 6. 利差分析与交易
- 信用利差 = 违约风险溢价 + 流动性溢价
- 与经济周期反向（扩张时压缩、衰退时走阔）
- Z-spread 剥离嵌入期权价值
- 信用利差曲线形态：正常斜率 > 0、平坦 / 倒挂为警示信号

### 7. 组合管理关键指标
- 市值加权平均久期、DV01、信用利差加权平均、评级分布、剩余期限梯度、现金流排期
- 再平衡频率：利率大幅变动 > 50bp 后立即调整 + 定期季度检查

### 8. 可转债三维估值
- **纯债价值（债底）**：Σ票息/(1+r)^t + 赎回价/(1+r)^n，85-100 常见
- **转股价值**：100 / 转股价 × 正股价
- **期权价值**：转债价 - max(纯债, 转股价)
- **双低策略**：价格 < 115 + 溢价率 < 30% → 双低值 < 130
- **下修风险**：触发条件 30 日连续 15 天 < 转股价 85%

## 建议骨架（不强制）

- 债券组合摘要（市值 / YTM / 修正久期 / DV01 / 平均评级）
- 利差分解表（政府 / 信用 / 流动性 / 残差）
- 信用风险评估（评级分布 / 城投占比 / 违约预期频率）
- 情景分析（利率 ±50/100/200bp、信用利差 ±50/100bp）
- 可转债持仓分析（三维估值 / 博弈概率 / 策略建议）

## 避坑

- 城投债评级普遍虚高（国内 AA 级约等国际 BBB-）——结合评级展望（正面 / 稳定 / 负面）和隐性债务指标
- 流动性溢价常被高估——小券深度不足容易自欺欺人，日成交额 < 500 万谨慎配置
- 可转债不要只看纯债价值就觉得安全——还需评估信用评级、票息递增的持续性、公司是否有动力下修
- 利差交易（做空高息债 + 做多国债）看似无风险——但高息债违约风险在压力时非线性跳升
- ABS / MBS 需穿透底层资产评估真实风险——不要仅看 tranche 优先级评级
