# 出海链投资（going-global）

> **何时参考**：用户问"出海概念 / 中国企业海外布局 / 国产替代到全球替代 / 哪些公司海外业务好 / 一带一路标的"。

## 核心目标

**评估中国企业的国际竞争力 + 海外成长兑现路径**——区分真出海（订单 + 利润）和题材出海。

## 分析步骤

1. **出海逻辑分类（不同类型估值体系不同）**：
   - **产品出口扩张**：周期短、相对易兑现，但竞争激烈
   - **海外产能布局**：capex 大、周期长 2-3 年、需评估东道国风险
   - **品牌全球化**：投入高、周期最长 5-10 年、难度最大
   - **技术 / 服务输出**：可复制性强但量级可能受限
2. **海外成长驱动拆解**：
   - 需求端：海外市场规模 / 替代进口 / 新品类渗透
   - 供给端：成本优势 15-30%（人力 / 管理 / 规模）/ 产能释放周期 / 交付能力
   - 国产替代 → 全球替代演进（光伏逆变器、锂电等）
   - **必须区分"能卖出去"和"能持续做大"**
3. **竞争力五维评估**：
   - 成本领先（制造业核心，15-30%）
   - 技术领先（高端芯片 / 医药 / 软件）
   - 产品力（消费品牌 / 快消）
   - 响应速度（大订单快速交付）
   - 客户资源与通道（to B 已建立的采购关系）
   真出海需至少一项硬优势，否则只是价格竞争（利润压力大）
4. **兑现路径分析**：
   - 订单获取机制（代理 / 经销商 / 直销 / 并购渠道）
   - 认证周期（欧盟 CE / 北美 FDA 等，6-18 个月）
   - 交付与生产（海外产能何时建成、代工 vs 自建）
   - 进口税 / 运费成本（额外 5-15%）
   - 汇率锁定（长期合同如何避免汇损）
   - 利润率路径（初期薄利 → 逐步积累议价权）
5. **地缘政治与贸易政策风险**：
   - 美国 301 关税（科技产品可能 25%）
   - 欧盟碳边界税（制造业 2-10% 额外成本）
   - 印太战略（芯片 / 半导体供应链风险）
   - 新兴市场货币贬值压力
6. **汇率与融资成本**：美元升值时出口收益性提升 → 反之亦然；新兴市场建厂面临货币贬值压力（巴西 / 印度 / 印尼）；长期出口合同需要汇率套保。
7. **文化与本地化适应**：品牌出海最容易栽跟头（缺乏当地认可度）；消费品需要本地化（口味 / 包装 / 营销语言）；B2B 相对容易（技术 + 成本说话）但售后服务关键。
8. **标的分层**：
   - **全球化龙头**：已在多地建厂、品牌知名、利润稳定
   - **正在突破二线**：订单高增、认证进行中、首次海外产能建设
   - **纯交易型主题**：概念股、无实质进展
   - **故事型弱兑现**：讲得好看但订单 / 进度无实质进展

## 建议骨架（不强制）

- 出海逻辑类型与估值框架
- 海外成长驱动分解
- 五维竞争力评分
- 兑现路径时间规划（订单 / 认证 / 产能 / 利润各阶段）
- 风险地图（贸易政策 / 汇率 / 地缘 / 本地化）
- 标的分层

## 避坑

- 不要被"海外需求旺盛"迷惑——能拿订单 ≠ 能盈利，汇率贬值与运费上升会快速侵蚀利润
- 海外产能建设周期常被严重低估——东南亚建厂通常需 2-3 年
- 美国关税变化波动大——301 清单 / 贸易协议都可能快速改变成本结构
- 新兴市场汇率贬值常在出海初期就发生
- 品牌全球化的投入与时间被极度低估——快消品在海外建立认可度需要 5-10 年 + 年均 1-2 亿营销投入
